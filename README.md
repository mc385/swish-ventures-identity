# Swish Ventures · Logo Identity System

Internal exploration & brand book for the Swish Ventures mark.

## What's inside

A Vite + React app that contains the entire identity exploration as browsable tabs:

- **Landing** — A15 cosmic landing page (Three.js particle assembly)
- **A15 Brandbook** — full 24-chapter brandbook for the chosen mark
- **Top 5** — final analysis: the five marks that survived, with scoring
- **Arch Plate** — exploration of the architectural plate direction (B0–D12 + ball studies)
- **Brandbook (v1)** — earlier brandbook around SV Lumina
- **Catalog** — first catalog of mockups
- **Exports** — every variant downloadable as SVG + PNG (512/1024/2048) with batch ZIP
- **Archive** — all earlier exploration directions for reference

## Run locally

```bash
npm install
npm run dev
```

The dev server starts at `http://localhost:5173` (or `:5174` if 5173 is in use).

## Build

```bash
npm run build
```

Output goes to `dist/`.

## Tech

- Vite + React 18
- Three.js (landing page particle system)
- JSZip (export bundles)
- Fonts: Fraunces, Inter, Cormorant Garamond, JetBrains Mono (Google Fonts)
